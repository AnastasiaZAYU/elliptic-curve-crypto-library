# Комп'ютерний практикум #1: Реалізація арифметики на еліптичних кривих
## Мета лабораторної роботи
Отримання практичних навичок програмної реалізації арфиметики на еліптичних кривих, закріплення теоретичного матеріалу, отриманого на лекційних заняттях.

## Завдання
1. **Реалізувати зручною мовою програмування бібліотеку функцій та типів** для роботи з еліптичними кривими:
   * Тип даних **"Еліптична крива"**;
   * Тип даних **"Точка еліптичної кривої"** (у афінних та проективних координатах);
   * Функції перевірки належності точки до кривої;
   * Функції додавання точок, подвоєння точки;
   * Функцію скалярного множення (з використанням одного з описаних алгоритмів);
   * Функції взаємного перетворення точки між афінними та проективними координатами.
3. **Реалізувати тестову програму**, що застосовує написані в п.1 функції для демонстрації арифметики над еліптичною кривою **P-224**.
4. **Перевірити коректність реалізації** арифметики: згенерувати випадкову точку P та перевірити, що виконується рівність nP=Oe, де n -- порядок кривої, а Oe -- точка на нескінченності.

## Деталі реалізації
Програма реалізована мовою програмування **Python** та складається з двох основних класів: `EllipticCurve` та `CurvePoint`.

### Клас EllipticCurve

Цей клас представляє еліптичну криву і містить наступні функції:

* `__init__(self, p, a, b, n=None)`: конструктор класу, що приймає параметри еліптичної кривої (p, a, b) та її порядок n.
* `IsOnCurve(curve, X, Y, Z=None)`: перевіряє, чи належить точка з координатами `(X, Y, Z)` заданій кривій `curve`.

### Клас CurvePoint

Цей клас представляє точку на еліптичній кривій та реалізує основні операції:

* `__init__(self, curve, X, Y, Z=None)`: конструктор класу, що приймає еліптичну криву, на якій лежить точка, та її координати `(X, Y, Z)`.
* `POINT_AT_INFINITY(Point)`: повертає точку на нескінченності.
* `ComparePoints(Point1, Point2)`: порівнює дві точки; повертає `True`, якщо вони ідентичні.
* `Print(Point)`: виводить координати заданої точки.
* `ToAffine(Point)`: перетворює точку з проективних координат в афінні.
* `ToProjective(Point)`: перетворює точку з афінних координат в проективні.
* `GenPoint(G)`: генерує випадкову точку на еліптичній кривій, використовуючи задану базову точку `G`.
* `PointDouble(Point)`: функція для подвоєння точки.
* `PointAdd(Point1, Point2)`: функція для додавання двох точок.
* `PointMultiply(Point, k)`: функція для скалярного множення точки на ціле число `k`.

**Файл EllipticCurves_lab1.ipynb** містить реалізацію тестової програми, яка демонструє застосування написаних функцій для арифметики над еліптичною кривою **P-224**. У ньому представлені приклади використання кожного методу та перевірка коректності виконання операцій.
